Judul Program : Mengurutkan umur dari yang paling muda  menggunakan Insertion Sort

Tujuan dari program ini adalah untuk membantu pengguna dalam menginput, menyimpan, dan mengurutkan data beberapa orang berdasarkan umur secara otomatis menggunakan algoritma Insertion Sort. Program ini dirancang agar pengguna dapat memasukkan nama dan umur dengan validasi input sehingga data yang dimasukkan lebih akurat. Setelah data dimasukkan, program akan menampilkan data sebelum dan sesudah proses pengurutan, sehingga memudahkan pengguna dalam melihat urutan umur dari yang termuda hingga tertua.
